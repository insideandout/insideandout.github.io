Learn Morse Code

Morse converts the each letter in the alphabet and each number in a series of dots and dashes to communicate. It was first used for radio transmission, and then they used morse code to send messages. There are two main types of morse code, which is the American, and the International. This page will help you learn the American version of the Morse code. You can click the page linked at the top to learn the International Morse code, and the other page will tell you a brief history of how the morse code came to be.
